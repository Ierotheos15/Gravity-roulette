# Gravity Roulette

Every minute, gravity picks a new direction. Could be up, down, north, south, east, or west. Good luck.

Made for Fabric 26.2 as part of HackCraft.

---

## What it does

Every 60 seconds the gravity direction changes for all players and mobs. You get a 1 minute headstart at the start before the first spin. The current direction and countdown are shown on your screen at all times.

If you're lucky, gravity stays down and nothing changes. If you're not, you're suddenly falling sideways into a wall.

- Never picks the same direction twice in a row
- Affects all mobs and animals too
- Works in all dimensions
- Spectator mode is not affected
- If you're under a block when gravity flips up, you're fine — that block is now your floor
- If there's nothing to stop you, you die

---

## Installation

1. Install [Fabric Loader](https://fabricmc.net/) for Minecraft 26.2
2. Download [Fabric API](https://modrinth.com/mod/fabric-api) for 26.2
3. Drop both `fabric-api.jar` and `gravity-roulette.jar` into your `mods/` folder
4. Launch the game

Works in singleplayer and multiplayer. For servers, both the server and all clients need the mod.

There are none. That's kind of the point.
